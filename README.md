# Dashboard · Parámetros de Calidad de Gasolinas
## UMSA · Ingeniería Petrolera · PET-212

Dashboard interactivo para el cálculo y análisis de parámetros de calidad de gasolinas base y mezclas con etanol.

### Funcionalidades
- Toma de muestras con 3 réplicas y estadística (media, σ, valor verdadero 95%)
- Cálculo de densidad — Fórmula GOST 3900 / ASTM D1217
- Corrección por temperatura con factor γ
- Punto Inicial de Ebullición (PIE) — Riazi + Clausius-Clapeyron
- RON estimado — Regla de mezclas volumétrica
- Verificación D.S. 4718 Bolivia
- Gráficas RON vs % EtOH y Densidad vs % EtOH
